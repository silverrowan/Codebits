# ICS 199 - Applied Computing Project
> [!info] 
>  **Credits::** 6
**Total Hours::** 175
**Restricted::** Information and Computer Systems, Interactive Media Developer

Are you ready to explore web application development and turn your innovative ideas into reality? In a collaborative environment, you will learn the process of defining, designing, developing, and implementing a simple yet impactful web application. Small teams will leverage collaboration and version control tools while navigating the challenges of project management. This course emphasizes continuous web integration and thorough technical documentation. You will learn industry skills but also essential teamwork communications and problem-solving abilities needed for effective web solutions.

> [!attention] 
> **Prerequisites::** [[ICS 124 - Algorithms & Data Structure Programming]] C, [[ICS 128 - Web Scripting]] C, [[ICS 120 - Database Concepts]] C, [[ICS 125 - Software Engineering Process]] C
